## 二、比赛提交版 API 口径

### 总提示
比赛提交版已经内置默认 API，评审或第一次试用时不需要先填写 Key。设置页仍然保留，但它是“可选设置”，不是核心引导的前置条件。

设置页只需要告诉用户三件事：

- 当前比赛包已经可以直接分析文章。

- 如果填写个人 API Key，不要发给别人，不要截图公开，也不要写进公开代码或公开文档。

## 四、分析结果页介绍

### 文章核心观点
这里是文章的核心观点。它帮你看清这篇文章到底在说什么，不只是记住标题。

### 文章结构
这里是结构复盘。它会告诉你文章是怎么开头、怎么推进、怎么收束的，帮你看懂为什么它能让人读下去。

### 情绪钩子
这里会帮你看到文章哪里在牵动读者情绪。以后你写类似内容时，可以回来参考这种进入方式。

### 爆点判断
这里会判断文章里最容易被读者记住、转发或讨论的地方。它不是夸张，而是帮你找到内容的力量点。

### 可复用素材
这里是素材库入口。好例子、好句子、好场景、好判断，以后都可以回来翻。

### 可迁移写法
这里是写法提醒。它会告诉你这篇文章里有哪些方法可以迁移到下一篇文章。

### 适合延展的选题
这里会给你下一步可以写什么。你不用每次从零开始想选题。

### 知识资产沉淀
这里会告诉你，这篇文章沉淀进了哪些领域、哪些节点、哪些标签。它已经不只是读过了，而是被放进你的知识图鉴里。

## 五、复盘模块介绍

复盘不是冷冰冰的报表，它更像有人陪你整理书桌。

你可以在这里看到：

- 最近分析过哪些文章。
- 哪些主题反复出现。
- 哪些素材还可以继续写。
- 哪些表达方式值得保留。
- 哪些内容已经变成自己的知识资产。
- 本周阅读和创作的变化。

气泡文案：
当你不知道自己最近到底积累了什么，就来复盘看看。它会帮你把散落的内容重新摆整齐。

按钮：
去看复盘

## 六、知识地图介绍

### 总说明
你每分析一篇文章，都会在这片知识宇宙里点亮一颗星星。

当你在某个领域写得越多、读得越多、积累得越深，那片星区就会越来越亮。一开始它可能只是一颗小点，后来会慢慢变成一片星云。

每个节点，都是你曾经认真理解过的一段内容。

### 引导点击星球
箭头指向：地图里的星星或星球。

气泡文案：
试着点一颗星星。点开以后，你会看到它背后的具体内容：来源文章、核心观点、标签、关联素材和复盘摘要。

这不是普通收藏夹。收藏夹只是把东西放进去，知识图鉴会帮你看见它们之间的关系。

按钮：
点一颗星星看看

## 七、模拟星球 / 完整版星球介绍

当一个领域被你持续点亮，星球会越来越完整。

如果你在某个主题下积累了很多文章、观点、素材和复盘，它就会像一颗被点亮的星球一样，出现更多小灯、节点和连接。

这些小灯不是装饰。每一盏灯都代表你在这个领域留下过一次理解。

气泡文案：
现在看到的是完整版星球的样子。当你的内容越来越多，它会越来越亮、越来越密，也越来越像你自己的知识地形。

按钮：
我明白星球是什么了

## 十二、开灯模式和关灯模式

箭头指向：顶部太阳 / 月亮按钮。

气泡文案：
这里可以切换开灯和关灯。

开灯模式更清楚，适合白天整理内容。蓝白色调会更像知乎，也更适合快速阅读。

关灯模式更沉浸，适合晚上复盘、看知识星球。关灯之后，你会更像是在看自己的知识宇宙，那些被你点亮的星星会显得更明显。

按钮：
切换看看
