# V2.1 T8 修正报告：宇宙二次视觉修正

时间：2026-05-08

## 本次问题

用户反馈上一版仍然丑，核心问题是：

- 知识星球太像大黑圆饼，不像星云里的小行星。
- 动效太快，不是“缓慢缓慢地变化”。
- 点击星球后的视角应该像被拉近星云，让选中星球成为主体。
- 交互应为：第一次点击锁定视角，第二次点击同一星球进入内部。

## 改动内容

### `panel.js`

- `panel.js:1687`
  - 收缩 `universeBallVisual()` 的星球尺寸：
    - 默认态从“大球”改成小型知识星球。
    - 仍保留“写得越多越大越亮”，但不再铺满画面。
- `panel.js:2013`
  - `lockUniverseBall()` 改为二段式：
    - 第一次点击：锁定该星球并拉近视角。
    - 再次点击同一星球：进入该星球内部单球视图。
- `panel.js:2059`
  - 更新提示文案，明确“再次点击同一星球进入内部”。
- `panel.js:2117`
  - `applyUniverseCameraLock()` 镜头缩放从过猛的 `3.08` 收到 `2.35`，让拉近明显但不至于把文字和星球压到失控。

### `panel.html`

- `panel.html:885`
  - 星云呼吸动画从 `18s` 改为 `160s`。
- `panel.html:889`
  - 银河旋转从 `145s` 改为 `900s`。
- `panel.html:923`
  - 星球光晕呼吸从 `5.8s` 改为 `28s`。
- `panel.html:917-920`
  - 锁定视角下，非目标星球降透明，且隐藏非目标星球文字，避免拉近后画面被无关标签污染。
- `panel.html:907`
  - 锁定视角下跨球虚线降低存在感，避免抢主体。

### `tests/panel_map.test.js`

- `tests/panel_map.test.js:199`
  - 镜头锁定断言更新为 `matrix(2.35...)`。

### `tests/universe_fixture.html`

- 同步真实面板的慢动画、锁定淡化、非目标标签隐藏样式。
