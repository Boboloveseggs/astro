# 第三步知乎引导强调样式完成报告

完成时间：2026-05-11

## 本次目标

调整面板新手引导第 3 步的文字强调：

- 标题里的“知乎”要金色、加粗、带更清楚的黑色压边感。

### 1. 第 3 步支持受控富文本

文件：`panel.js`

- 给“分析第一篇文章前，先打开知乎正文”这一步新增：
  - `titleHtml`
  - `bodyHtml`
- `renderPanelOnboarding()` 支持：
  - 有 `titleHtml` 时用 `innerHTML`
  - 有 `bodyHtml` 时用 `innerHTML`
  - 其他步骤仍然使用原来的纯文本逻辑

### 2. 增加引导强调样式

文件：`panel.html`

- 新增 `.onboarding-em-gold`
  - 金色
  - 加粗
  - 黑色压边/暗色阴影
- 新增 `.onboarding-em-strong`
  - 加粗加重
  - 深色模式和开灯模式都有对应可读颜色

### 3. 增加防回归测试

文件：`tests/panel_cleanup.test.js`

- 新增断言：
  - 第 3 步标题里的“知乎”必须有 `onboarding-em-gold`
