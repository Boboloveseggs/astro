# V21 T8 演示星球匿名铺点完成报告

生成时间：2026-05-09

## 本次目标

为了方便给其他用户演示，在知识星球里增加一个“演示星球”模式：即使当前真实节点很少，也能用匿名发光点铺满整个球面，让它一眼看起来是完整的知识星球。

## 改动内容

1. `panel.js`
   - 新增 `earthSphereDemoFillEnabled` 演示开关。
   - 新增 `EARTH_SPHERE_DEMO_POINT_COUNT = 180`，用 Fibonacci 球面分布生成匿名演示点。
   - 新增 `buildEarthSphereDemoPoints()`：只生成视觉点，不带真实节点名称。
   - 新增 `renderEarthSphereDemoFill()`：渲染匿名发光点层。
   - 新增 `syncEarthSphereDemoVisibility()`：演示模式开启时隐藏真实节点层和连线层，关闭后恢复。
   - 新增工具栏按钮 `演示星球 / 关闭演示`。
   - 演示点接入原有球面投影、拖拽旋转、自动旋转、重置视角和缩放逻辑。

2. `panel.html`
   - 新增 `.sphere-demo-node` / `.sphere-demo-core` / `.sphere-demo-glow` 样式。
   - 演示点禁用 pointer events，不参与 hover、点击和真实节点交互。

3. `tests/earth_sphere_fixture.html`
   - 补齐演示点样式。
   - 支持 `?demo=1` 自动打开演示模式，方便截图验收。

4. `tests/panel_map.test.js`
   - 新增 1 个回归测试：
     - 默认不显示演示点。
     - 点击“演示星球”后渲染 180 个匿名点。
     - 真实节点和连线在演示模式下隐藏。
     - 演示点不暴露真实节点名称。
     - 演示点会随球体旋转，不是静态贴图。
     - 关闭演示后真实节点和连线恢复。

## 验收标准

1. 稀疏节点数据也能显示为完整球体。
2. 演示模式不展示具体节点名称、节点标签和连线。
3. 演示点只作为视觉层，不写入数据库、不修改 schema、不改变 Markdown 格式。
4. 演示点跟随知识球旋转、缩放和重置。
5. 关闭演示模式后恢复真实知识球。
6. 自动化测试全通过。
