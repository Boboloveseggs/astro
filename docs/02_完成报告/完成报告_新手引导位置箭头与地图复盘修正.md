# 新手引导位置箭头与地图复盘修正完成报告

完成时间：2026-05-10

## 本次目标

修正面板新手引导的三个体验问题：

- 引导块太靠上，介绍复盘/时间轴时遮挡后面的内容。
- 地图引导反复切换时容易让宇宙/星球视角重置，像“撞墙”。
- 箭头位置不标准，指向不准。

## 实际改动

### 1. 引导目标改为真实内容区

文件：`panel.js`

- 复盘里的「创作方向 / 高频节点 / 时间轴 / AI 复盘」不再指向上方小导航按钮。
- 改为指向当前复盘分页面板里的真实标题：
  - `#review-direction .review-section-title`
  - `#review-nodes .review-section-title`
  - `#review-timeline .review-section-title`
  - `#review-chat .review-section-title`

### 2. 地图引导不再高亮整个宇宙容器

文件：`panel.js`

- 删除地图引导对 `#mapTab` 大容器的指向。
- 改为指向：
  - `#mapHint`
  - `.universe-tools`
- 文案补充“点一次锁定，再点同一颗进入内部”，避免用户在引导中连续点星球导致视角乱跳。

### 3. 防止地图引导重复重载宇宙

文件：`panel.js`

- 新增 `navigatePanelOnboardingStep()`。
- 如果当前已经在地图 Tab，后续地图引导步骤不会再次调用 `switchTab('map')`。
- 这样不会在每个地图引导步骤里重复 `loadMap({ resetToOverview: true })`。

### 4. 引导卡片改成目标感知定位

文件：`panel.js`、`panel.html`

- 新增 `highlightPanelOnboardingTarget()`、`findPanelOnboardingTarget()`。
- 新增动态 `positionOnboardingArrow()`：
  - 根据目标位置自动把卡片放到目标上方或下方。
  - 卡片不再固定堆在顶部。
  - 箭头会根据卡片位置显示 `↑` 或 `↓`。
- 浮层背景变轻，卡片最大高度从 `48vh` 收敛为 `42vh`，减少遮挡。
- 卡片移动加入轻微过渡，避免跳动感。

### 5. 增加防回归测试

文件：`tests/panel_cleanup.test.js`

- 新增断言：复盘细分引导必须指向当前分页面板，不应只指导航按钮。
- 新增断言：地图引导不能再指向 `#mapTab` 整个容器。

## 回滚路径

如需回滚本次修改：

1. 将 `panel.js` 中复盘目标恢复为 `.review-anchor[data-anchor="..."]`。
2. 将地图引导目标恢复为旧逻辑前的目标。
3. 删除 `navigatePanelOnboardingStep()`、`findPanelOnboardingTarget()`、`highlightPanelOnboardingTarget()`、新版 `positionOnboardingArrow()`。
4. 恢复 `panel.html` 中 `.onboarding-card` 的旧高度与无动态定位样式。
5. 删除 `tests/panel_cleanup.test.js` 中本次新增的两条断言。
