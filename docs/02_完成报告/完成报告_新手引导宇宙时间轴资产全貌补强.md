# 新手引导宇宙时间轴资产全貌补强完成报告

完成时间：2026-05-11

## 本次目标

修正新手引导里几个“不够到位”的地方：

- 宇宙介绍太短，没把这块用心做的核心体验讲出来。
- “点一颗星星”步骤箭头指到了底部提示，不是宇宙画面。
- 时间轴、地图这类大模块只介绍开头，用户看不到全貌。
- 资产页需要明确告诉用户：每次分析完成的完整报告在“作品”里找。

## 实际改动

### 1. 资产页新增“作品”指示步骤

文件：`panel.js`

- 在资产引导后新增一步：
  - 标题：`完整分析报告，在“作品”里找`
  - 目标：`.asset-mode[data-asset-mode="assets"]`
- 文案说明：
  - 完整分析报告在“作品”里。
  - 素材和立意是从报告里拆出来的可复用片段。

同时调整引导切到资产页时的加载逻辑，确保资产工具栏会渲染出来，箭头能找到“作品”按钮。

### 2. 时间轴新增“亮背景看全貌”步骤

文件：`panel.js`、`panel.html`

- 在“时间轴：看见你的积累顺序”后新增一步：
  - 标题：`先看一眼时间轴全貌`
  - 目标：`#review-timeline`
  - `peek: true`
- 这一类步骤会让遮罩变轻、背景亮起来，用户先看清整块面板长什么样，再继续下一步。

### 3. 宇宙介绍补强

文件：`panel.js`

- “地图：这里是你的知识宇宙”文案扩写。
- 强调：
  - 这里不是列表，也不是普通收藏夹。
  - 它会把文章、主题、节点和项目慢慢变成知识宇宙。
  - 每篇文章都会点亮一点光，积累越深，星区越亮。

### 4. 地图新增“亮背景看宇宙全貌”步骤

文件：`panel.js`、`panel.html`

- 在地图介绍后新增一步：
  - 标题：`先看一眼宇宙全貌`
  - 目标：`#universeCanvasWrap`
  - `peek: true`
- 文案说明：
  - 中间缓慢旋转的星云是归档宇宙。
  - 小亮点是知识星球。
  - 主题积累越多，星星越亮。

### 5. 修正“点一颗星星”箭头目标

文件：`panel.js`

- “点一颗星星，看它背后的内容”目标从 `#mapHint` 改为 `#universeCanvasWrap`。
- 箭头现在指向宇宙画面，不再指向底部提示。
- 文案补充：
  - 鼠标移到星星上会出现名字。
  - 点一次锁定视角，再点同一颗进入内部。
  - 不要连续急点，等镜头停稳后再操作。

### 6. 新增 `peek` 引导模式

文件：`panel.js`、`panel.html`

- `renderPanelOnboarding()` 会根据步骤的 `peek: true` 给浮层加 `.peek`。
- `.onboarding-layer.peek` 会把背景遮罩变轻，并取消模糊。
- 用在需要“先看全貌”的步骤，不影响普通步骤。

### 7. 增加防回归测试

文件：`tests/panel_cleanup.test.js`

新增断言：

- 资产引导必须说明完整报告在“作品”里查找。
- 时间轴必须有亮背景全貌观察步。
- 地图必须有亮背景宇宙全貌观察步。
- “点一颗星星”步骤目标必须是 `#universeCanvasWrap`，不能再指向底部提示。
