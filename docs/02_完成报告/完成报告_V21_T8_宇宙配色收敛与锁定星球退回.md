# 完成报告：V21 T8 宇宙配色收敛与锁定星球退回

## 本次目标

继续修正归档宇宙页：

- 去掉“花花绿绿”的杂色感，统一成紫黑金。
- 锁定/拉近知识星球后，不再只是一个发光点，而是切换成有明暗、纹理、边缘的星球贴图。
- 鼠标滚轮往后缩时，自动从锁定星球退回全景宇宙，而不是只把近景缩小。

## 实际改动

### `panel.js`

- 新增 `universePlanetTexture()`：
  - 生成 512px 程序星球贴图。
  - 包含球面明暗、紫金表面纹理、淡金边缘和弱大气。
  - 用于锁定状态的知识星球，避免继续表现成单纯光球。
- 收敛星云色彩：
  - `universeGalaxyTexture()` 的冷色粒子从蓝青倾向改成黑紫/暗紫。
  - `createUniversePointCloud()` 的远星、haze、dust、bright 冷色全部改为紫系。
  - `createUniverseSpiralArmCloud()` 标记 `palette: 'purple-black-gold'`，并统一成紫黑金粒子。
  - 跨星球连线改成暗紫色，避免蓝线破坏主色。
- 锁定星球视觉：
  - 锁定对象 `material.map` 切到 `planetTexture`。
  - 锁定对象使用 `THREE.NormalBlending`，不再是纯加色发光点。
  - 锁定对象提高 `renderOrder`，让星球主体浮在星云背景上。
  - 锁定状态下压低背景星云、粒子层和核心光晕透明度，避免星球被中心星尘吞掉。
- 锁定镜头：
  - 点击/锁定星球时目标缩放改为 `0.68`，比全景更近，但不直接把星球压满画面。
  - 镜头 `lookAt` 改为锁定星球自身，减少目标偏到画面边缘的问题。
- 滚轮退回全景：
  - 新增 `releaseUniverseLockToOverview()`。
  - 锁定状态下滚轮向后缩放时，自动清空 `activeBallId / lockedBall / lockedObject`。
  - 同时触发回到全景宇宙的镜头 tween，并保存地图状态。
  - “解除锁定”按钮也复用同一逻辑。

### `tests/panel_map.test.js`

- 增加断言：
  - 星云底图声明 `palette === 'purple-black-gold'`。
  - `planetTexture.userData.kind === 'planet'`。
  - `planetTexture.userData.palette === 'purple-black-gold'`。
  - 螺旋粒子层声明紫黑金调色。
  - 锁定对象使用 `planetTexture`。
  - 锁定状态下滚轮向后会回到全景，并清空 `activeBallId`。

## 风险与后续建议

- 当前星球贴图是 2D billboard，不是真正 Three.js 球体；优点是轻、稳定，缺点是极近距离下不会有真实 3D 自转。
- 如果后续还要更像“进入一个真实星球”，建议下一轮把锁定态对象升级成小型 `SphereGeometry`，材质用程序 CanvasTexture，再在二次点击时切到知识球内部视图。
