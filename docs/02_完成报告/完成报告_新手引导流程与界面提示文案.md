# 完成报告：新手引导流程与界面提示文案

完成时间：2026-05-10

## 本次目标

为知识图鉴设计一套完整的新手引导流程和界面提示文案，把完全不懂电脑的新用户当作第一次接触插件的人，逐步引导他完成：

- 认识产品；
- 设置 API Key；
- 第一次分析知乎文章；
- 理解分析结果；
- 查看资产、复盘、地图、星球；
- 学会分析下一篇文章；
- 固定插件；
- 切换开灯 / 关灯模式。

## 交付内容

新增文档：

```text
新手引导流程与界面提示文案.md
```

文档包含：

- 欢迎页文案；
- API Key 九步设置引导；
- 第一次分析文章流程；
- 分析结果页逐项气泡；
- 复盘模块介绍；
- 知识地图介绍；
- 模拟星球 / 完整版星球介绍；
- 总复盘报告介绍；
- 目标达成设置说明；
- 下一篇文章分析说明；
- 插件固定到浏览器说明；
- 开灯 / 关灯模式说明；
- 21 步完整新手引导气泡文案。

## 文案原则

- 使用温柔、陪伴式语气；
- 尽量像界面气泡，不写成冷冰冰说明书；
- 每一步都告诉用户看哪里、点哪里、点完会发生什么；
- API Key 明确提醒用户自己创建、自己保存，不要发给别人，不要公开截图，不要写进公开代码；
- 文章处理统一使用「读取当前页面内容」「分析当前文章」等表达；
- 强化知识宇宙、星星、星球、节点逐渐点亮的产品画面感。
