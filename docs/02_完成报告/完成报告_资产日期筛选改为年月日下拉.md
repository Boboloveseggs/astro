# 完成报告：资产日期筛选改为年月日下拉

时间：2026-05-10 11:35

## 本次问题

上一版用了浏览器原生 `type=date` 日期框，会弹系统日历，视觉不可控，也不符合“直接选择年月日”的需求。

## 本次改动

1. `panel.js:1385-1459`
   - 移除原生日历框。
   - 新增 `assetDateSelectHtml()`，渲染三个下拉：
     - 年
     - 月
     - 日
   - 新增筛选辅助函数：
     - `getAssetDateFilterParts()`
     - `hasAssetDateFilter()`
     - `assetDateFilterLabel()`
     - `matchesAssetDateFilter()`
     - `updateAssetDateFilterOptions()`

2. `panel.js:1574, 1594`
   - 工具栏改为使用年月日下拉。
   - 三个下拉任意变化都会重新加载当前资产列表。

3. `panel.js:1655-1666`
   - 素材/立意按年月日下拉筛选。
   - 只选年可看整年，只选年月可看整月，年月日都选则精确到天。

4. `panel.js:1711-1724`
   - 作品/速记按年月日下拉筛选。

5. `panel.html:1472-1480, 1691-1700`
   - 新增年月日下拉样式，并接入现有主题输入框样式。

6. `tests/panel_assets.test.js:38-41, 233`
   - 工具栏测试改为确认存在年/月/日三个下拉。
   - 日期筛选测试改为用年月日下拉筛选作品。
