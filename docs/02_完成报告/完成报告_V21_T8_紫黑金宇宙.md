## 1.0 引用与复刻说明

本次宇宙球继续复用 T2/T3 修正后的 1.0 视觉系统：

- 领域球核心填充：`url(#rg-active)`，沿用深棕渐变。
- 球体描边：`var(--gold)` / hover 后 `var(--gold-light)`。

- 发光滤镜：`ballGlowFilter` 与单球 `glow-filter` 保持同类暗金发光逻辑。

没有恢复之前被否定的“四种节点类型金色调色板”，也没有恢复实心大球外轮廓。

## 风险与后续

- `zhj_map_state` 是 UI 状态，不进入 JSON 备份，也不影响老数据。
- 15 个领域球在小面板内仍可能有轻微标签拥挤，但比初版明显分散；后续可在 T8 打磨里做避让布局。
- project 模式会读取全库文章按项目分球，这符合“宇宙视角”而不是当前项目内视角。
