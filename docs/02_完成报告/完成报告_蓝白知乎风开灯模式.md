# 完成报告：蓝白知乎风开灯模式

完成时间：2026-05-10 19:28

## 本次目标

增加一个「开灯模式」，以蓝白色调和知乎风格为主。默认仍保留原来的黑金风，用户可以自己切换。

## 改动内容

1. `panel.html`
   - 调整 `html.theme-light` 变量为蓝白知乎风：
     - 主色：`#1772f6`
     - 背景：白蓝渐变
     - 卡片：白色 / 浅蓝白
     - 边框：低饱和蓝色
   - 顶部齿轮旁新增开灯按钮 `themeToggleBtn`。
   - 暗色时显示太阳 `☀`，点击进入开灯模式；开灯后显示月亮 `☾`，点击回到暗色。

2. `panel.js`
   - 新增主题状态：
     - `UI_THEME_KEY = 'uiTheme'`
     - `applyUiTheme()`
     - `loadUiTheme()`
     - `toggleUiTheme()`
   - 主题保存到 `chrome.storage.local.uiTheme`。
   - 打开 panel 时自动读取上次主题。
   - 监听主题变化，多个页面之间可以同步。

3. `options.html`
   - 设置页同步使用同一套蓝白开灯变量，避免打开设置页时突然回到黑金。

4. `options.js`
   - 设置页读取 `uiTheme` 并应用。
   - 监听主题变化。

5. `tests/panel_cleanup.test.js`
   - 新增测试：验证开灯模式会添加 `theme-light`，按钮文案 / 图标状态会同步更新。
