# V21 地图页空间释放与知识球缩放增强完成报告

生成时间：2026-05-09

## 本次目标

仅修改地图 / 知识星球这一页：取消顶部“总作品 / 总字数 / 本周进度”栏块，顶部尽量只保留项目选择和设置入口，把空间让给知识星球和宇宙；同时放开知识球放大比例，支持更近距离观察节点。

## 改动内容

1. `panel.html`
   - 新增 `body[data-active-tab="map"]` 专用样式。
   - 地图页隐藏：
     - 顶部标题 `h1`
     - `.top-metrics` 总作品 / 总字数 / 本周进度栏
   - 地图页保留：
     - 项目选择栏
     - 删除项目按钮
     - 设置按钮
     - 原有 tab 导航
   - 地图页压缩外边距：
     - body padding 从默认 `20px 16px` 调整为地图态 `10px 10px`
     - header 间距缩小
     - 项目栏最大宽度放开
   - 地图舞台高度增加：
     - `.graph-area` 最小高度提升为 `max(540px, calc(100vh - 126px))`
     - `.universe-area` 最小高度提升为 `max(580px, calc(100vh - 126px))`
     - `#earthSphere` / `#universeView` 同步增加高度

2. `panel.js`
   - `switchTab()` 中写入 `document.body.dataset.activeTab = name`，让 CSS 只在地图页生效。
   - 初始化时默认设置 `data-active-tab="analyze"`。
   - 新增知识球缩放常量：
     - `EARTH_SPHERE_MIN_ZOOM = 0.62`
     - `EARTH_SPHERE_MAX_ZOOM = 2.35`
     - `EARTH_SPHERE_WHEEL_ZOOM_STEP = 0.11`
     - `EARTH_SPHERE_BUTTON_ZOOM_STEP = 0.16`
   - `setEarthSphereZoom()` 改用新上下限，放大上限从原来的 `1.22` 提升到 `2.35`。
   - 放大 / 缩小按钮和滚轮步进同步提高。

3. `tests/panel_map.test.js`
   - 新增缩放边界测试：
     - 放大不能超过 `EARTH_SPHERE_MAX_ZOOM`
     - 缩小不能低于 `EARTH_SPHERE_MIN_ZOOM`
     - 回到中心后恢复 `EARTH_SPHERE_INITIAL_ZOOM`
