# 完成报告：新手引导浮层实现

完成时间：2026-05-10 20:02

## 本次目标

把新手引导真正做进插件里，而不是只写文档。首次打开面板时，用户会看到一步一步的引导浮层；点击去设置 API Key 后，设置页会继续接上 API Key 填写引导。

## 改动内容

1. `panel.html`
   - 新增新手引导浮层 DOM：
     - `onboardingLayer`
     - `onboardingArrow`
     - `onboardingMeta`
     - `onboardingTitle`
     - `onboardingBody`
     - 上一步 / 下一步 / 先跳过按钮
   - 新增浮层样式：
     - 半透明遮罩；
     - 当前目标高亮；
     - 箭头提示；
     - 温柔的卡片式说明。

2. `panel.js`
   - 新增面板引导状态：
     - `onboardingComplete`
     - `onboardingOptionsPending`
   - 新增 `panelOnboardingSteps()`，覆盖：
     - 欢迎；
     - API Key 设置入口；
     - 第一次分析文章；
     - 重新读取当前文章按钮；
     - 开始分析；
     - 分析结果；
     - 资产；
     - 复盘；
     - 地图；
     - 星球；
     - 总复盘报告；
     - 目标达成；
     - 下一篇文章；
     - 固定插件；
     - 开灯 / 关灯。
   - 新增引导控制函数：
     - `maybeStartPanelOnboarding()`
     - `startPanelOnboarding()`
     - `renderPanelOnboarding()`
     - `nextPanelOnboarding()`
     - `backPanelOnboarding()`
     - `finishPanelOnboarding()`
   - 引导按钮会真的切换到资产、复盘、地图、总复盘等页面。
   - 点击「去设置 API Key」会打开设置页，并给设置页写入继续引导标记。

3. `options.html`
   - 新增设置页 API Key 引导浮层 DOM 和样式。

4. `options.js`
   - 新增 `optionsOnboardingSteps()`，覆盖：
     - 设置页说明；
     - AI 提供商；
     - 官方平台创建 API Key；
     - 粘贴 API Key；
     - 保存设置；
     - 返回使用。
   - 支持从面板跳转过来后自动启动设置页引导。
   - 设置页引导结束后清除 `onboardingOptionsPending` 标记。

5. `tests/panel_cleanup.test.js`
   - 新增测试确认面板新手引导覆盖设置、分析、资产、复盘、地图、总复盘和开关灯。
