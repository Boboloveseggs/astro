# V21 T8 归档宇宙全景与镜头推入完成报告

生成时间：2026-05-09

## 本次目标

把归档 / 地图页面的宇宙改得更接近参考图：第一屏是铺满画面的紫黑金旋涡星系，上面的亮点就是知识星球；点击亮点后，镜头从全景丝滑推入到中景，像一镜到底进入宇宙，而不是生硬切换。

## 改动内容

1. `panel.js`
   - `universeGalaxyTexture()` 改为更大的 `1600 x 1000` 纹理。
   - 新增 5600 个大半径 haze 粒子，形成更厚的紫黑金星云外臂。
   - 主体星云粒子从 7200 提升到 11600，并扩大旋涡半径。
   - 核心粒子从 1800 提升到 2600，中心更亮。
   - 星云平面从 `760 x 470` 提升到 `1180 x 750`，让归档宇宙铺满画面。
   - `universe3DPoint()` 重新分布知识星球：
     - 星球分布到更大的旋臂范围。
     - 由原来的紧凑中心布局改成沿星云散布。
   - 知识星球 sprite 调亮、放大，并设置 `depthTest: false`，保证它们作为可点击星球压在星云之上。
   - 全景相机从 `42°` 调整为 `46°`，更适合展示大星系。
   - 点击锁定镜头时长提升到 `3000ms`，强化一镜到底的推入感。
   - 锁定镜头目标改为斜向贴近星球，保留中景，不做硬切换。
   - 星云俯仰从 `-0.44` 调整为 `-0.30`，减少被压成一条带的问题。

2. 验收截图
   - 全景宇宙：`screenshots_V21_T8_archive_universe/archive_universe_spiral_15balls.png`
   - 锁定中景：`screenshots_V21_T8_archive_universe/archive_universe_locked_midshot_v2.png`

1. 归档页宇宙不再缩成中心一小坨。

4. 点击星球后用相机缓动推入，不直接切页面。
5. 第二次点击同一星球仍进入内部金色知识球。
6. 不改数据库 schema。
7. 不改 Markdown 格式。
8. 自动化测试全通过。
