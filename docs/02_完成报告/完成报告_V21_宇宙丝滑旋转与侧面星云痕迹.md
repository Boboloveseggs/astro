# 完成报告：V21 宇宙丝滑旋转与侧面星云痕迹

## 本次目标

继续修正归档宇宙页：

- 拖拽转动不够丝滑，应该有平滑追随感。
- 侧面视角不能变成乱散点，应该仍能看到星云走势和螺旋痕迹。

## 实际改动

### `panel.js`

- 新增 `smoothUniverseAngle()`：
  - 拖拽不再直接把画面旋转到最终角度。
  - 改为 `targetRotation` 接收拖拽目标，`userRotation` 每帧平滑追随。
  - 见 `panel.js:1713`、`panel.js:3146-3167`、`panel.js:3226-3229`。
- 新增 `targetRotation` 状态：
  - 保留真实旋转值和目标旋转值，减少拖拽时的机械跳动。
  - 见 `panel.js:3458`。
- 新增 `createUniverseRibbonCloud()`：
  - 生成约 `9800` 个沿螺旋路径分布的 3D 星云丝带粒子。
  - 丝带有明确 z 轴起伏，侧翻时仍能看到星云轨迹。
  - 见 `panel.js:2898`。
- 接入 `ribbonCloud`：
  - `ribbonCloud` 加入 `galaxyGroup` 和 `particleClouds`。
  - 见 `panel.js:3367`、`panel.js:3375`、`panel.js:3446-3448`。
- 降低随机 halo 噪声：
  - `halo` 透明度从 `0.34` 降为 `0.24`，避免侧翻时随机粒子压过星云骨架。

### `tests/panel_map.test.js`

- 新增断言：必须存在 `ribbonCloud` 且粒子数不少于 `9000`，见 `tests/panel_map.test.js:218`。
- 新增断言：`ribbonCloud` 的 z 轴纵深必须超过 `360`，见 `tests/panel_map.test.js:224`。
- 新增断言：拖拽使用 `targetRotation`，真实旋转不能瞬间跳到目标角度，见 `tests/panel_map.test.js:243-253`。
