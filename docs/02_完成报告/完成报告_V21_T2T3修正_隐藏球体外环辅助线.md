# V2.1 T2/T3 修正补丁：隐藏球体外环辅助线

时间：2026-05-08

## 本次问题

球体外侧仍渲染了一圈外轮廓和经纬辅助线，视觉上形成明显边框，破坏 1.0 暗金球体的通透感。

## 改动内容

### `panel.js`

- `panel.js:1487`
  - `renderSphereShell(layer)` 改为只清空 `#sphereShell`。
  - 不再生成外轮廓 `circle`。
  - 不再生成纬线 / 经线 `ellipse`。

### `panel.html`

- `panel.html:855`
  - `.sphere-shell-line` / `.sphere-shell-lat` 设为 `display: none`。
  - 即使旧 DOM 或测试夹具残留该 class，也不会显示外环辅助线。

### `tests/earth_sphere_fixture.html`

- `tests/earth_sphere_fixture.html:126`
  - 同步隐藏 `.sphere-shell-line` / `.sphere-shell-lat`，确保截图夹具与真实面板一致。

### `tests/panel_map.test.js`

- `tests/panel_map.test.js:82`
  - 新增断言：`#sphereShell` 不再有子元素。
- `tests/panel_map.test.js:83`
  - 新增断言：DOM 中不再出现 `.sphere-shell-line`。
