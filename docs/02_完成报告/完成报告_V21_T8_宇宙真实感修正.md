# V2.1 T8 修正报告：宇宙真实感与星球锁定

时间：2026-05-08

## 本次问题

上一版 T8 虽然实现了“按领域分球”，但视觉更像黑紫画布上摆了几颗球，不像用户给出的银河宇宙参考图。用户明确要求：

- 背景要像真正的宇宙 / 银河。
- 里面有很多知识星球。
- 哪个星球写得越多，就越大、越亮。
- 找不到自己的星球时，可以通过顶部项目栏自动锁定。
- 常态要缓慢运行，有呼吸感、移动感、旋转感。
- 选中星球后要像球体锁定一样，视角发生变化。

## 改动内容

### `panel.js`

- `panel.js:1501`
  - 强化 `universeDefsHtml()`：新增银河核心、蓝色银河盘、星球 aura 等 SVG 渐变。
- `panel.js:1614`
  - 重写 `universePoint()`：领域球不再平铺在普通圆形区域，而是按银河倾角分布到旋臂附近。
- `panel.js:1644`
  - `renderStars()` 从约 30 个星点升级为默认 320+ 个密集星场。
- `panel.js:1662`
  - 新增 `renderGalaxyDust()`：沿旋臂生成蓝白 / 金色尘埃星点。
- `panel.js:1687`
  - 新增 `universeBallVisual()`：根据 `articleCount` 计算星球半径、光晕半径、光晕透明度、描边强度。
- `panel.js:1699`
  - 新增 `resolveUniverseLockBallId()`：
    - 如果已有锁定星球且仍存在，继续锁定。
    - project 模式下优先锁定顶部项目栏当前项目。
    - domain 模式下，按顶部项目栏当前项目的主领域自动锁定对应领域球。
- `panel.js:1712`
  - `renderUniverseBalls()` 改为：
    - 使用 `universeBallVisual()` 控制“写得越多越大越亮”。
    - locked 星球最后渲染，避免被其他星球盖住。
    - 单击星球锁定视角，双击进入该星球单球视图。
- `panel.js:2013`
  - 新增 `lockUniverseBall()`：单击知识星球后只锁定宇宙视角，不直接跳转。
- `panel.js:2033`
  - `renderUniverseView()` 改为真正银河结构：
    - `#universeMotion`
    - `#universeCamera`
    - `#galaxyLayer`
    - `#starsLayer`
    - `#galaxyDustLayer`
    - `#ballLinksLayer`
    - `#ballsLayer`
  - 工具栏改为“进入星球 / 解除锁定 / 按领域 / 按项目”。
  - 锁定后提示当前锁定星球。
- `panel.js:2113`
  - 新增 `applyUniverseCameraLock()`：锁定星球后将镜头矩阵放大并平移到该星球。

### `panel.html`

- `panel.html:870-926`
  - 强化宇宙视觉：
    - `.universe-area` 更深的宇宙背景和内阴影。

    - `#galaxyLayer` 使用 `galaxySlowTurn` 做极慢旋转。
    - `.universe-ball-glow` 使用 `planetPulse` 做星球呼吸光晕。
    - 锁定时非目标星球降透明，目标星球保持高亮。

  - `renderUniverseView`：确认银河层、密集星场、旋臂尘埃星点、锁定星球和镜头矩阵存在。

### `tests/universe_fixture.html`
