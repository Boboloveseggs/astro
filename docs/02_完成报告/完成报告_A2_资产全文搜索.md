# A2 完成报告 · 资产全文搜索

## 做了什么

### panel.html
- 新增 `.asset-search` 样式（深色输入框，聚焦时边框变色，占满工具栏剩余空间）
- 调整 `.assets-toolbar-btn` 和 `.sync-status` 加 `flex-shrink:0`，防止搜索框把按钮挤压

### panel.js · `initAssetsToolbar()`
- 工具栏现在**始终**渲染搜索输入框（不再依赖是否绑定文件夹）
- 重新扫描按钮仅在检测到本地文件夹时追加到搜索框右侧
- 搜索框 `input` 事件直接触发 `loadAssets()`，无防抖（资产量不大，实时更合适）

### panel.js · `loadAssets()`
- 每次调用时读取 `#assetSearch` 的当前值作为查询词
- 过滤字段：文章标题、核心判断、主领域、子领域、作者、编辑评语（全部不区分大小写）
- 空结果时显示「没有找到「xxx」相关的记录」，无搜索词时显示原来的空状态提示

## 验收标准

| 步骤 | 预期结果 |
|------|----------|
| 切到「资产」Tab | 顶部出现搜索框（始终可见），有绑定文件夹时搜索框右侧显示「重新扫描」按钮 |
| 在搜索框输入关键词 | 列表实时过滤，只显示标题/核心判断/领域包含该词的条目 |
| 清空搜索框 | 恢复显示全部记录 |
| 搜索无结果 | 显示「没有找到「xxx」相关的记录」 |
| 切换项目或删除记录后 | 重新调用 `loadAssets()` 时仍保留搜索框现有文字并重新过滤 |
