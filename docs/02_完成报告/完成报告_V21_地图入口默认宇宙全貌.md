# 完成报告：地图入口默认宇宙全貌

时间：2026-05-09

## 本次改动

- `panel.js`
  - `switchTab('map')` 进入地图时改为调用 `loadMap({ resetToOverview: true })`。
  - `loadMap(options)` 增加 `resetToOverview` 入口参数：从顶部 Tab 点击地图时强制进入宇宙全貌，并清空上次锁定的 `activeBallId`。
  - `resolveUniverseLockBallId()` 取消按当前项目/主领域自动锁定知识球，只保留用户显式选择的 `activeBallId`。

- `tests/panel_map.test.js`
  - 更新锁定逻辑测试：默认不自动锁定；只有显式 `activeBallId` 存在且匹配时才锁定。
  - 更新 3D 宇宙渲染测试：先验收默认宇宙全貌，再显式选择知识球验收锁定预览。

## 验收标准

- 点击“地图”时，默认看到宇宙全貌，而不是某一个具体知识球。
- 之前点过某个知识球，也不会在下次重新进入地图时自动恢复锁定。
- 用户在宇宙里明确点击某个知识球后，仍然可以进入锁定预览，再次点击进入内部知识球。
- 不改宇宙颜色、旋转参数、螺旋臂、星云粒子密度等视觉参数。

## 实际结果

- 已符合：地图入口会清空 `activeBallId` 并显示宇宙全貌。
- 已符合：显式选择知识球后仍保留锁定预览能力。
- 已符合：本次没有改动宇宙视觉参数，只改入口状态与自动锁定逻辑。
