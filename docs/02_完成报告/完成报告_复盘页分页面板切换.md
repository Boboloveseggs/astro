# 完成报告：复盘页分页面板切换

时间：2026-05-10 10:53

## 本次问题

复盘页把「本周概览 / 创作方向 / 高频节点 / 时间轴 / AI 复盘」全部堆在同一个长页面里，点击导航时会快速自动滚动到下方区块，视觉上很乱。

## 本次改动

1. `panel.js:63`
   - 新增 `activeReviewSection`，记录复盘页当前打开的二级页面。

2. `panel.js:4118-4161`
   - `reviewAnchorsHtml()` 支持按当前页面高亮入口。
   - `reviewNavBlockHtml()` 支持传入当前页面。
   - `bindReviewAnchors()` 从 `scrollIntoView` 锚点滚动改为切换 `activeReviewSection` 后重新渲染。
   - 删除复盘页导航里的 `IntersectionObserver` 自动跟随滚动逻辑。

3. `panel.js:5549-5567`
   - `loadReview()` 改为一次只渲染一个复盘分区。
   - 「时间轴」只在时间轴页绑定 deck 交互。
   - 「AI 复盘」只在 AI 复盘页绑定对话交互。
   - 「高频节点」只在高频节点页绑定展开收起。
   - 「创作方向」只在创作方向页加载 AI 引导。

4. `tests/panel_review.test.js:173`
   - 更新复盘导航测试：确认 5 个入口仍存在，但页面只渲染当前分区，不再同时堆叠 5 个 section。
