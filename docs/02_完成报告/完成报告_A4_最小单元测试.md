## 做了什么

### db.js 顺带修复
- `parseMarkdownRecord` 补充了 `tags` 字段的解析（之前 A3 写入 md 但没有读回），保证往返一致性

### tests/runner.js
- 极简测试框架：`test(name, fn)`、`assert(cond, msg)`、`assertEqual(a, b, msg)`、`Suite.run()`
- 支持异步测试（async fn）
- 测试结果逐条 DOM 渲染，✓绿色 ✗红色，底部汇总 pass/fail 数

### tests/test.html
- 深色主题测试页，依次加载 runner.js → ../db.js → ../analyzer.js → db.test.js → analyzer.test.js → 自动运行
- 打开方式：在 Chrome 扩展管理页找到扩展 ID，访问 `chrome-extension://[ID]/tests/test.html`
