## 节点位置算法说明

本次使用 Fibonacci sphere：

```js
function sphereVectorForSeed(name, total, slot) {
  const golden = Math.PI * (3 - Math.sqrt(5));
  const t = (slot + 0.5) / Math.max(1, total);
  const lat = Math.asin(1 - 2 * t) * 180 / Math.PI;
  const lon = ((slot * golden) * 180 / Math.PI) % 360 - 180;
  return spherePoint(lat, lon);
}
```

它不是手工经纬度表，而是按黄金角把点均匀排到单位球面。节点按确定性顺序排序后再分配 slot，所以同一批节点位置稳定；节点数变化时重新计算，避免 1.0 手工点位导致的局部重叠。

## 亮度 5 等级映射表

| 等级 | 条件示意 | 半径 | 透明度 | 外发光 |
|---|---|---:|---:|---|
| 1 | 1 篇 | 4 | 0.40 | 无 |
| 2 | 2-3 篇 | 5 | 0.55 | 无 |
| 3 | 4-6 篇 | 6 | 0.70 | 弱 |
| 4 | 7-10 篇 | 8 | 0.85 | 中 |
| 5 | 11 篇以上，或高频 primary | 10 | 1.00 | 强，双层金色发光 |

补充规则：`role === 'primary'` 时等级 +1，上限 5；球体背面节点整体透明度降低。
