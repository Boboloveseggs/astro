# 完成报告：新手引导首次出现与复盘细化修正

完成时间：2026-05-10 20:12

## 问题

用户反馈三点：

1. 新手引导应当只在下载安装后第一次出现，或第一次打开知识图鉴时出现，不希望每次都出现。
2. 引导卡片放在底部会和地图 / 星球的节点区域重合，容易误触。
3. 复盘页引导太粗，只说了复盘，没有介绍创作方向、高频节点、时间轴、AI 复盘。

## 修复内容

1. `panel.js`
   - 新增 `ONBOARDING_SEEN_KEY = 'onboardingSeen'`。
   - `maybeStartPanelOnboarding()` 改为：
     - 如果 `onboardingComplete` 或 `onboardingSeen` 任意一个存在，就不自动弹出；
     - 第一次弹出时立刻写入 `onboardingSeen: true`；
     - 即使用户中途关闭或没走完，下次也不会再自动出现。
   - `finishPanelOnboarding()` 同时写入：
     - `onboardingComplete: true`
     - `onboardingSeen: true`

2. `panel.html`
   - 新手引导浮层从底部移动到上方：
     - `align-items: flex-start`
     - `padding: 72px 18px 18px`
   - 引导卡片增加最大高度和滚动：
     - 避免长文案撑满界面；
     - 避免压住底部地图 / 星球操作区。

3. `options.html`
   - 设置页 API Key 引导也同步改为上方卡片布局。

4. `panel.js` 复盘引导细化
   - 在原复盘引导后新增 4 个独立步骤：
     - 创作方向；
     - 高频节点；
     - 时间轴；
     - AI 复盘。
   - 每一步会设置 `reviewSection`，切到复盘页对应子页面，而不是只讲一个笼统的复盘。

5. `tests/panel_cleanup.test.js`
   - 新增断言确认引导包含：
     - `reviewSection === 'direction'`
     - `reviewSection === 'nodes'`
     - `reviewSection === 'timeline'`
     - `reviewSection === 'chat'`
