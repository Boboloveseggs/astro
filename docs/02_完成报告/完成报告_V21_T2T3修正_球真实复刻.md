# V2.1 T2/T3 修正完成报告：球真实复刻 1.0

执行依据：`任务下发_V21_T2T3修正_球真实复刻1.0.md`。  
修正目标：作废前一版「四金色节点 + 球面渐变填充」方向，恢复 1.0 的「深色画布 + 鎏金小石头」视觉。

## 做了什么

1. 删除错误方向：
   - 删除 `EARTH_NODE_PALETTE`。
   - 删除 `earthTypeColor()`。
   - 删除 `earthSphereBody / earthSphereGloss` 球面填充渐变。
   - 删除 `earthSphereBodyLayer / earthSphereGlossLayer` 球面填充层。
2. 复刻 1.0 节点视觉：
   - 新增 `rg-active` 深棕激活态节点渐变。
   - 新增 `rg-locked` 更深棕锁定态节点渐变。
   - 新增 `earthNodeFill()`，按 `lv` 选择 active / locked。
   - 新增 `earthNodeStrokeColor()`，按亮度等级使用 `gold-light / gold / gold-dim` 描边。
   - 新增 `earthNodeStrokeWidth()`，按亮度等级调整描边粗细。
   - 4/5 级节点额外加金色外晕，5 级节点使用 `glow-filter`。
3. 复刻 1.0 球感：
   - 使用 `rg-glow` + `mapGlow` 椭圆中心光晕。
   - 球面不再被金色填充覆盖，背景透出。
   - 经纬线恢复为弱金色辅助线：`opacity 0.32 / 0.24`。
4. 连线分级：
   - 新增 `linkClass()`。
   - 共现 3 次以上为 `sphere-link-strong`。
   - 共现 1 次为 `sphere-link-mid`。
   - `wikilink` 或 value 0 为 `sphere-link-weak` 虚线。
5. 图例修正：
   - 从「人物/概念/理论/事件」类型色图例，改为「强连接 / 弱连接 / 待点亮」连线图例。
   - 节点类型仍在 hover 卡和资产详情中显示，不丢语义。
6. 同步更新截图 fixture，生成 5/30/100 节点截图。
