# 知识图鉴 2.0 Demo 录制脚本

目标时长：3 分钟

## 0:00-0:30 定位

开场句：

> 知识图鉴 2.0 是一个给持续创作者使用的个人作品整理工具。它不是通用笔记本，而是把我写过的文章自动分析、归档、连成地图，并反过来指导下一次创作。

画面：

1. 打开扩展侧边栏。
2. 展示顶部项目选择、总作品、总字数、本周进度。
3. 说明痛点：写过很多内容，但很难知道自己到底在反复写什么、哪些素材能复用、下一篇该往哪里走。

## 0:30-1:30 抓取与分析

画面：

1. 在知乎文章页点击扩展图标。
2. 侧边栏识别文章标题。
3. 点击「开始分析」。
4. 分析完成后展示 5 组结果：
   - 定位
   - 创作者反馈
   - 素材资产
   - 延伸方向
   - 追问 AI

解说重点：

> 2.0 的分析不是只判断文章好不好，而是站在创作者角度拆解：我这篇文章的优势是什么，写作技法在哪里，可复用素材是什么，立意能不能沉淀下来。

特写：

1. 「创作者优势」卡片。
2. 「写作技法」卡片。
3. 「可复用素材」和「立意精华」卡片。

## 1:30-2:00 追问与入库

画面：

1. 在「追问 AI」里点击「我这篇最好的地方是什么？」。
2. 等待 AI 返回。
3. 展示回答下方「已自动更新立意库」。
4. 切到资产 Tab，进入「立意」子分类，看到刚刚沉淀的内容。

解说重点：

> 分析不是终点。追问会把文章里的经验继续提炼成素材或立意，自动进入资产库。以后找素材不是按标题翻，而是按创作角度找。

## 2:00-2:30 地图与时间轴

画面：

1. 切到地图 Tab。
2. 展示 3D 地球图谱，拖动球体。
3. 停留节点 1 秒，展示 hover 卡里的关联文章。
4. 切到复盘 Tab 的时间轴牌组。
5. 拖动底部 slider 或点击右侧日期 gear。

解说重点：

> 地图解决“我写过哪些知识节点”的问题，时间轴解决“我的创作是怎么展开的”的问题。一个看关系，一个看节奏。

## 2:30-3:00 周复盘

画面：

1. 在复盘 Tab 顶部展示本周作品、本周字数、日均频率、月度对比。
2. 展示「创作方向分析」：已点亮方向、尚未出现的视角、已建立的写作模式。
3. 点击「打开 AI 复盘对话」。
4. 点击模板「下周该尝试什么新方向？」。

解说收束：

> 最终，知识图鉴 2.0 把“一篇文章的分析”变成“长期创作的反馈系统”：抓取、分析、追问、入库、地图、时间轴、周复盘，全部围绕我自己的作品展开。

结束画面：

> 知识图鉴 2.0：给创作者自己的知识地图和作品复盘系统。
