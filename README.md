# 知乎创作图鉴 2.0

知乎创作图鉴 2.0 是一个面向知乎创作者的作品资产库、写作指纹系统与个人知识宇宙，用来把自己的知乎文章、手动粘贴内容和速记沉淀为可搜索、可追问、可继续创作的个人作品图鉴。

比赛版主线：让创作者过去写过的每一篇知乎内容，变成下一次创作可调用的观点资产。

## 核心能力

- 一键抓取知乎内容或手动粘贴文章。
- AI 分析作品资产定位、创作者优势判断、写作指纹证据、可复用素材、立意精华、知识节点和下一篇创作方向。
- 分析结果会优先展示 6 类作品资产卡：核心观点、知识节点、可复用素材、延展方向、写作指纹证据、创作者优势判断。
- 每张作品资产卡都可以“用于新文章”，把旧观点、旧素材和旧节点继续生成下一篇选题、标题、开头、大纲、反驳点与结尾。
- “读出的你”会从长期作品里生成写作指纹：高频主题、论证方式、表达气质、反复世界观、问题切口和作品之间的隐性连接。
- 资产库搜索会按相关观点、相关素材、相关知识节点、写作指纹、可复用金句和可延展方向找回旧资产，让旧文章继续为下一篇创作工作。
- “分身”Tab 提供“问问过去的我”：基于历史作品资产回答问题，引用旧文、观点卡、素材卡和知识节点。
- 追问 AI：6 个单篇追问模板，自定义追问，并可沉淀到素材库或立意库。
- 资产库：按素材、立意、作品浏览。
- 3D 知识地图：用球面节点展示作品中的知识节点关系，点击节点可查看关联文章、关联资产、写作指纹意义和下一篇延展方向。
- 时间轴牌组：按日期浏览作品演化。
- 复盘 Tab：查看本周作品、本周字数、日均频率、月度对比、创作方向分析，并可打开 AI 周复盘对话。
- 本地 Markdown 文件夹同步与 Obsidian Vault 兼容导出保留在设置页高级能力中。

## 本地运行

1. 打开 Chrome 扩展管理页。
2. 开启「开发者模式」。
3. 选择「加载已解压的扩展程序」。
4. 比赛提交版选择解压后的 `astro.release` 文件夹；开发版选择 `zhijing_v4_competition` 文件夹。
5. 比赛提交版已内置默认 AI 分析通道，可以直接打开侧边栏分析文章；设置页只用于可选更换自己的 API、模型或知乎开放平台增强配置。

## 评审试用路径

1. 打开知乎创作图鉴侧边栏。
2. 按引导载入示例文章。
3. 点击「开始分析」。
4. 等待作品资产卡生成，并确认文章已归库。
5. 之后再按需点击「资产」「复盘」「地图」「读出的你」「分身」等区域；每个区域会在首次进入时展示对应引导。
