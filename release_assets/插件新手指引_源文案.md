# 插件新手指引
知乎创作图鉴浏览器扩展安装说明

这份文档只教你一件事：把压缩包里的插件顺利放进浏览器。插件打开后的内部使用教程，已经放在面板右上角的问号里。你可以自由选择看不看。

## 安装前先看
- 先把压缩包解压，不要直接在压缩包里安装。
- 打开浏览器右上角的扩展管理页：Chrome 可以输入 chrome://extensions，Edge 可以输入 edge://extensions。
- 打开“开发者模式”。这个开关通常在扩展管理页右上角或左侧。
- 点击“加载已解压的扩展程序”，然后选择文件夹“知乎创作图鉴插件”。
- 安装完成后，把插件固定到浏览器右上角，后面分析文章会更方便。

## API Key 安全提醒
API Key 必须由你自己在官方平台创建，只填写在知乎创作图鉴的设置页里。不要把 API Key 发给别人，不要截图公开，也不要写进公开代码。

## 第一次打开插件后
- 如果你是第一次空库使用，插件会自动带入一组演示文章，让知识宇宙和知识星球先亮起来。
- 面板右上角有问号按钮，需要详细教程时点它，它会一步一步带你设置 API Key、分析知乎文章、查看资产、地图和总复盘报告。
- 真正开始使用时，请打开你想分析的知乎文章页面，等正文完整显示，再点击“开始分析”。

## 安装完成后怎么开始
- 打开自己的知乎主页，挑一篇想整理的文章。
- 点进文章详情页，等正文完整显示。
- 点击知乎创作图鉴插件，选择项目，点击“开始分析”。
- 分析完成后，文章会进入资产里的“作品”；素材和立意会进入对应库；地图里会点亮一颗星。
- 每周可以打开“总复盘报告”，看看最近积累了哪些主题、素材和写法。

如果你看不懂某一步，不用急。打开插件后点问号，跟着面板上的引导一步一步点就可以。
