# 插件新手指引（参赛版）

知乎创作图鉴浏览器扩展安装说明。

这份文档只教你一件事：把压缩包里的插件顺利放进浏览器。插件打开后的内部使用教程，已经放在面板右上角的问号里。

## 安装前先看

- 先把压缩包解压，不要直接在压缩包里安装。
- Chrome 可以在地址栏输入 `chrome://extensions`，Edge 可以输入 `edge://extensions`。
- 打开“开发者模式”。这个开关通常在扩展管理页右上角或左侧。
- 点击“加载已解压的扩展程序”，然后选择解压后的插件文件夹。
- 安装完成后，把知乎创作图鉴固定到浏览器右上角，后面分析文章会更方便。

## 默认 API 说明

参赛版已经默认内置可用 API，安装后不需要先去设置 Key。设置页只是备用入口：以后如果想换成自己的模型或 Key，再去那里修改即可。

## 第一次打开插件后

- 第一次体验时，直接打开插件，按引导点击“开始分析”，插件会用准备好的内容走完分析与归库。
- 首次引导只介绍核心功能：分析一篇文章、查看归库结果。
- 地图、复盘、作者分身等功能，会在你点到对应区域时再出现对应说明。

## 正式使用

- 打开你想分析的知乎文章页面，等正文完整显示。
- 点击知乎创作图鉴插件，点击“开始分析”。
- 分析完成后，文章会进入作品资产库，地图里会点亮对应节点。

如果你看不懂某一步，不用急。打开插件后点问号，跟着面板上的引导一步一步点就可以；一般不需要设置 API。
